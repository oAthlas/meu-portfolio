Projeto em Destaque: Lilie

Lilie – Assistente Virtual com IA

A Lilie é uma assistente pessoal criada com Python e inteligência artificial. Ela se conecta ao OpenRouter para gerar respostas inteligentes, realiza agendamentos com a Google Calendar API e pode fazer pesquisas na web automaticamente com a API de busca do Google.

A Lilie representa meu primeiro grande projeto, desenvolvido com foco em resolver problemas reais e oferecer uma experiência fluida de interação homem-máquina.
